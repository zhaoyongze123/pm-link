# 详细初始化指南

> RuoYi-Vue-Pro 工作流与审批系统精简版

## 第一步：环境检查

### 检查 Java 版本
```bash
java -version
# 应该是 Java 8 或更高版本
```

### 检查 Maven 版本
```bash
mvn -version
# 应该是 3.6.0 或更高版本
```

### 检查 Node.js（前端开发）
```bash
node -v
npm -v
# 应该是 16.0 以上
```

## 第二步：数据库初始化

### 1. 创建数据库
```bash
# MySQL 示例
mysql -u root -p
CREATE DATABASE ruoyi_vue_pro DEFAULT CHARSET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. 导入 SQL 脚本
```bash
# MySQL 示例
mysql -u root -p ruoyi_vue_pro < repo/sql/mysql/ruoyi-vue-pro-mysql8.sql
```

### 3. 配置数据库连接
编辑 `repo/yudao-server/src/main/resources/application-local.yaml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/ruoyi_vue_pro
    username: root
    password: your_password
    driver-class-name: com.mysql.cj.jdbc.Driver
```

## 第三步：启动后端

### 方式 1：使用 IDEA

1. 打开 IDEA
2. 打开项目：`/Users/mac/项目/若伊部署/repo`
3. 等待 Maven 索引完成
4. 右键点击项目 → Maven → Reload Projects
5. 找到 `YudaoServerApplication.java`
6. 点击 Run 按钮或按 Ctrl+R

### 方式 2：命令行启动

```bash
cd /Users/mac/项目/若伊部署/repo
mvn spring-boot:run -pl yudao-server
```

### 验证后端启动

访问：http://localhost:8080

应该看到登录页面。

## 第四步：启动前端

### 进入前端目录
```bash
cd /Users/mac/项目/若伊部署/repo/yudao-ui/yudao-ui-admin-vue3
```

### 安装依赖
```bash
npm install
# 或使用 pnpm（更快）
```

### 启动开发服务
```bash

```

### 访问前端
打开浏览器访问输出的地址，通常是：http://localhost:5173

## 常见问题排查

### 后端启动失败

**错误信息**：端口被占用
```bash
# 杀死占用 8080 端口的进程
lsof -i :8080
kill -9 <PID>
```

**错误信息**：数据库连接失败
- 检查数据库是否启动
- 检查数据库连接信息是否正确
- 检查 SQL 脚本是否导入

### 前端启动失败

**错误信息**：node_modules 问题
```bash
rm -rf node_modules package-lock.json
npm install
```

**错误信息**：PORT 被占用
```bash
# 修改 vite.config.ts 中的端口
# 或杀死占用该端口的进程
```

### 工作流菜单不显示

1. 确认 SQL 脚本是否导入完成
2. 重启后端服务
3. 清空浏览器缓存
4. 检查用户权限

## 常用命令

### Maven 命令
```bash
# 清理编译
mvn clean

# 编译
mvn compile

# 打包
mvn package

# 跳过测试打包
mvn clean package -DskipTests

# 刷新依赖
mvn clean install -DskipTests

# 仅编译服务器
mvn clean package -pl yudao-server -DskipTests
```

### 启动脚本
```bash
# 启动后端
bash 脚本/快速启动.sh backend

# 启动前端
bash 脚本/快速启动.sh frontend

# 启动全部
bash 脚本/快速启动.sh all
```

### 数据库命令
```bash
# MySQL 导入
mysql -u root -p database < file.sql

# MySQL 导出
mysqldump -u root -p database > backup.sql
```

## 数据库恢复

如果数据库出问题，可以恢复初始状态：

```bash
# 删除数据库
mysql -u root -p -e "DROP DATABASE ruoyi_vue_pro;"

# 创建数据库
mysql -u root -p -e "CREATE DATABASE ruoyi_vue_pro CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 重新导入 SQL 脚本
mysql -u root -p ruoyi_vue_pro < repo/sql/mysql/ruoyi-vue-pro-mysql8.sql
```
