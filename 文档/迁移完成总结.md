### 方式二：手动启动

#### 1. 后端启动
```bash
cd /Users/mac/项目/若伊部署/repo

# 刷新依赖
mvn clean install -DskipTests

# 启动应用
mvn spring-boot:run -pl yudao-server
```

#### 2. 前端启动
```bash
cd /Users/mac/项目/若伊部署/repo/yudao-ui/yudao-ui-admin-vue3

# 安装依赖
npm install

# 启动开发服务
npm run dev
```

## ⚠️ 常见问题

### Q1: 后续如何添加其他模块？
**A**: 非常简单：
1. 在 `repo/pom.xml` 中取消对应模块的注释
2. 在 `repo/yudao-server/pom.xml` 中取消对应依赖的注释
3. 执行 Maven 刷新：`mvn clean install`
4. 重启后端服务

### Q2: 工作流菜单没有显示？
**A**: 检查清单：
1. ✅ 导入了 BPM SQL 脚本
2. ✅ 重启了后端服务
3. ✅ 登录用户有 BPM 模块权限

### Q3: 前端启动失败？
**A**: 清理并重新安装：
```bash
rm -rf node_modules package-lock.json
npm install
```
