🚀 快速开始

方式一：使用启动脚本（推荐）
  bash 文档/脚本/快速启动.sh backend   # 启动后端
  bash 文档/脚本/快速启动.sh frontend  # 启动前端
  bash 文档/脚本/快速启动.sh all       # 启动全部

方式二：手动启动
  后端：cd repo && mvn spring-boot:run -pl yudao-server
  前端：cd repo/yudao-ui/yudao-ui-admin-vue3 && npm install && npm run dev

访问地址：
  后端：http://localhost:8080
  前端：http://localhost:5173
  用户：admin
